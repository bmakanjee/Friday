### Source: "Monarch Operations Brain" (832 messages, largest conversation)

#### Micro-SOP System (M01-M06+)
A complete micro-SOP framework was developed with these modules:
- **M01A/B** - Top-Down Wash (1-Man / 2-Man) using foam cannon
- **M02A/B** - Wheel & Tyre Clean
- **M03** - Exterior Drying
- **M04** - Spray Sealant / Drying Aid
- **M05** - Interior Vacuum
- **M06A/B** - Dash/Console/Door Wipe
- Additional SOPs planned: M07 Cup Holders & Crevices, M08 Mat Shampoo & Extraction

#### SOP Format (finalised)
Each micro-SOP follows a simplified worker-ready format:
1. Objective (one sentence, plain English)
2. Tools You Need (icons/colour codes only)
3. Steps (1-Man) - very detailed, one action per line
4. Steps (2-Man) - clear split: "Tech A / Tech B"
5. Checks Before You Finish - simple checklist

**Key decision:** No dilution ratios inside SOPs (Braj pre-mixes all bottles). SOPs reference bottle colours only.

#### Colour Bottle System
| Bottle Colour | Use |
|---------------|-----|
| Blue | Foam Gun (Carnauba / Ceramic Shampoo) |
| Brown | Wheel Cleaner |
| Yellow | Multi-Purpose Cleaner (G101) |
| Green | Glass Cleaner |
| Pink | Interior Dressing (Finish) |

#### Wash Method Decision
- Foam cannon replaces 2-bucket method as standard
- No buckets in daily use except as backup or water reserve
- Carnauba Shield Plus = standard shampoo
- Ceramic Shampoo = premium upsell only (not every job)
- AquaSeal SiO2 as drying aid = the "wow finish" step

#### G101 Usage Decision
G101 NOT used on exterior as standard. Reserved for:
- Door jambs
- Bug spots
- Very dirty areas
- Interior cleaning

## 2. Dilution Ratios (Master Reference)

### Source: "Monarch Operations Brain"

All chemicals are AutoSmart brand, purchased in bulk (20L/5L).

| Chemical | Use | Dilution |
|----------|-----|----------|
| Carnauba Shield Plus (20L) | Standard wash & wax | Bucket: 30mL per 10L. Foam gun: 1:10 |
| Ceramic Shampoo | Premium/upsell wash | Bucket: 25-30mL per 10L. Foam gun: 1:10 |
| Wheel King (20L) | Wheels/tyres | 1:4 (heavy), 1:8 (normal), 1:10 (light) |
| G101 (20L) | APC - pre-spray, door jambs, interior | 1:10 heavy, 1:20 general, 1:30 light |
| Smart Glass (5L) | All glass (ammonia-free, tint-safe) | Ready to use (or 1:1 for light films) |
| Finish (5L) | Interior dressing (satin, water-based) | Neat or 1:1 |
| Quick Shine (5L) | Tyre/exterior trims (gloss, solvent-based) | Neat |
| Trim Ultra (1L) | Faded exterior plastics restorer | Neat |
| Autofresh ICE (5L) | Air freshener | Ready to use (light mist only) |
| AquaSeal SiO2 | Spray sealant / drying aid | As per label |

**Key distinction:**
- Finish = interior (satin/low sheen)
- Quick Shine = tyres (gloss/durable)

### Pre-Diluted Bottle System
- Van carries 1L pre-diluted bottles for foam cannon
- Minimum 5x Wash'n'Wax + 5x Ceramic Shampoo ready at all times
- Stored in vertical slots with front lip, non-slip mat, labelled + dated

- Don't stack Wheel King + Brake Buster wet-on-wet (dilutes/neutralises)
- IK Foamer recommended for applying wheel chemicals (creates foam/cling)
- Wheel King diluted 1:10 in IK foamer for normal use
