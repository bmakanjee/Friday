# PULSE — Reporting Learnings

## Key business numbers
- Break-even: 47 jobs/month (9.4/week)
- Current pace: ~6 jobs/week = ~$3,400/month loss
- Monthly fixed costs: ~$7,000 (van $1,000 + detailer ~$4,600 + fuel/supplies)
- Average job price: $200
- Ad budget: $500-1,000/month (currently $300/month at $10/day)

## Reporting preferences
- Telegram briefs: 5-8 lines max, lead with what changed
- Desktop reports: full detail with tables
- Always include comparison period (vs yesterday, vs last week)
- Every data point needs an action implication — no data dumps
- Australian English, AUD currency, AEST timezone
- Round financials to 2dp, percentages to 1dp
